# ຮຽນພາສາເກົາຫຼີ ສຳລັບແຮງງານລະດູການ

ເວັບໄຊ້ຮຽນພາສາເກົາຫຼີ-ລາວ ສຳລັບແຮງງານລະດູການລາວ, ດັດແປງຈາກປຶ້ມຄູ່ມື "ພາສາເກົາຫຼີ ສຳລັບແຮງງານລະດູການລາວ".

ເປັນເວັບໄຊ້ static ລ້ວນໆ (HTML/CSS/JS), ບໍ່ຕ້ອງມີ server ຫຼື database — ຂໍ້ມູນຄວາມຄືບໜ້າ/ຄະແນນເກັບໄວ້ໃນ browser ຂອງແຕ່ລະຄົນ (localStorage).

## ໂຄງສ້າງ

- `index.html`, `styles.css`, `app.js` — ໂຕແອັບ
- `data.js` — ເນື້ອຫາບົດຮຽນທັງໝົດ (5 ບົດ, 610 ຄຳສັບ/ປະໂຫຍກ)

## ຟີເຈີ

- 5 ບົດຮຽນ: ການທັກທາຍ, ຊີວິດປະຈຳວັນ, ການເຮັດວຽກ, ເຄື່ອງມືກະສິກຳ, ອາການເຈັບປ່ວຍ/ສຸກເສີນ
- ແຕ່ລະຄຳ: ຄຳອ່ານພາສາລາວ + ໂຕໜັງສືເກົາຫຼີ + ຄວາມໝາຍ + ສຽງອ່ານ (Text-to-Speech)
- ແບບທົດສອບ 4 ໂຕເລືອກ ຫຼັງແຕ່ລະບົດ (ຜ່ານ ≥70%)
- ລົງທະບຽນຊື່ນັກຮຽນ ແລະ ຕິດຕາມຄະແນນ
- ໜ້າແອັດມິນ (ລະຫັດ PIN): ເບິ່ງລາຍຊື່ນັກຮຽນ/ຄະແນນ, ແກ້ໄຂເນື້ອຫາບົດຮຽນ, ດາວໂຫຼດ/ນຳເຂົ້າຂໍ້ມູນ

## ນຳໃຊ້ແບບ Offline (ໃນຄອມພິວເຕີ)

```bash
python3 -m http.server 8080
```
ແລ້ວເປີດ `http://localhost:8080`
